# PacmanTS

### Live link
https://wahrechrist.github.io/pacmnts/

### Overview
Classic pacman game written completely on Typescript/Vanilla JS

### Features

- Score
- Win/Lose state
- Usage of vanilla JS syntax only, without any frameworks
- Compiled with Vite
